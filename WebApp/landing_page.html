Welcome to ParlaStats

Welcome to ParlaStats, a statistical tool designed for the analysis of parliamentary discourse from various countries, primarily within Europe. ParlaStats is built upon the multilingual and comparable corpora of the ParlaMint project, with an extension for Czech parliamentary data through the enhanced ParCzech corpus.

ParlaStats has been developed as a bachelor's thesis at Charles University.

Author: Matěj Bačo
Supervisors: Barbora Hladká,Matyáš Kopp

Current capabilities

At its current stage of development, ParlaStats supports the analysis of the enhanced Czech ParCzech corpus, as well as the Slovenian and Ukrainian corpora from the ParlaMint project.

You can explore these datasets through the querying interface accessible via the "Tools" tab in the menu. This interface provides a form wher you can specify (1) What you are searching for, (2) Which conditions the results should meet, and (3) How the data should be grouped and sorted.

Once you created and submitted your query, the results will be displayed in a tabular format, which also allows you to export the data as an "Excel-like" CSV file, see Tools > RESULTS TABLE.

In addition to tables, results are also visualized using bar (primary) and line charts, see Tools > RESULTS GRAPH. These charts support interactive features, such zooming and the ability to toggle data visibility by clicking items in the chart's legend.

Feedback and Improvement

As mentioned earlier, ParlaStats is being developed as part of my bachelor's thesis and is intended to serve as the foundation for my future Master's thesis. As such, it's still a work in progress. It may not always function perfectly, might crash unexpectedly or produce incorrect results. While I do my best to identify and fix these issues, the occasional bug can still slip through. If you encounter any problems, especially if ParlaStats breaks under your queries, please let me know at matejbaco95@gmail.com or submit an issue to GitHub

When submitting an error report, it would be extremely helpful if you could also include the contents of the "Manual Query" tab (found next to the main query form). It may look intimidating, but it provides crucial technical details that help me track down the issue faster. Feel free to add colorful commentary, but please include that data!

If you are not intimidated by the "Manual Query" contents, feel free to experiment using debug mode. For guidance, check out the "Tutorial" section in the menu.
